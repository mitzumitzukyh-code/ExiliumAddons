# Content Agents Skill

Este directorio contiene los archivos de skill/prompt para flujos de trabajo de Content Agents.

## Estructura

- `SKILL.md`: definición principal de la skill.
- `content-agents-SKILL.md`: variante/documentación adicional de la skill.
- `PROMPT_WINDSURF.md`: prompt asociado para el flujo Windsurf.

## Uso sugerido

1. Revisa `SKILL.md` como fuente principal.
2. Usa `PROMPT_WINDSURF.md` cuando quieras ejecutar el flujo de prompt de Windsurf.
3. Mantén sincronizados `SKILL.md` y `content-agents-SKILL.md` si haces cambios funcionales.

## Notas

- Se movió esta skill a `docs/content-agents-skill` para mantener separados los assets de documentación del código del addon.
